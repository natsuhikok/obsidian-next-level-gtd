# Next Level GTD

Next Level GTD is an Obsidian plugin for practicing GTD (Getting Things Done) inside your vault.

It helps you:

- classify notes as Inbox, Reference, or Actionable
- track actionable notes with a small set of statuses
- review unclassified or problematic notes in one place
- see your available next actions across the vault

## What It Does

The plugin adds a lightweight GTD layer on top of regular Markdown notes.

Core capabilities:

- note classification through frontmatter
- status switching from the note UI
- next action detection from Markdown checkboxes
- Inbox view for triage and alerts
- Next Actions view for execution
- folder exclusion and vault initialization settings

## Typical Workflow

1. Capture notes in Obsidian as usual.
2. Classify each note as Reference or Actionable.
3. Add checkbox tasks to actionable notes.
4. Use the Inbox view to process unclassified notes.
5. Use the Next Actions view to focus on available work.

## Commands

The plugin provides commands for opening the main views and updating the active note's state.

It also includes commands for:

- changing status
- opening the Inbox view
- opening the Next Actions view
- cancelling all next actions in the active note

## Settings

Available settings include:

- excluded folders
- vault initialization for existing notes
- development helpers for mock data
